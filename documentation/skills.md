# Required Skills & Capabilities

## 1. Frontend Engineering
*   **Canvas Manipulation:** Expertise in PixiJS, Fabric.js, or Konva for building the visual editor.
*   **Performance Optimization:** Handling large assets (4k images, video) in the browser without lag.
*   **State Management:** Complex undo/redo stacks, optimistic UI updates.

## 2. AI Engineering
*   **Prompt Engineering:** Ability to craft "System Prompts" that take raw user intent and enforce brand guidelines.
*   **API Integration:**
    *   **Image Gen:** Stable Diffusion (Auto1111 / ComfyUI API), Midjourney (if API available), DALL-E 3.
    *   **LLM:** OpenAI (GPT-4), Anthropic (Claude 3).
*   **LoRA Management:** Understanding of how to train and serve LoRA adapters for specific visual styles.

## 3. Design System Implementation
*   **Atomic Design:** Building a scalable component library from scratch.
*   **Headless UI:** Implementing accessible components (Radix UI / React Aria) with custom styling.

## 4. Backend & DevOps
*   **Asset Processing:** FFmpeg for video, Sharp/ImageMagick for images.
*   **Queue Management:** Handling long-running AI generation tasks (BullMQ / Redis).
